Randomize

The Randomize menu can be found with the [Randomize...] button in the main General Tool menu. This operation allows you to place blocks randomly from a source area.

Select Source​

If you have not selected a source for getting random blocks yet, the [▶Run] button will be grayed out. The [Select Source...] button opens a menu that allows you to select two corner positions of the area you'd like to have blocks taken from and used in the Randomize operation.

You can select a corner of the source area by either right-clicking a block with the General Tool to place it there, or using the [Set Position Here] button to place it at your player position.

After both positions have been set, you have to option to either select [Exclude...] and then choose which blocks to ignore from the source region, or select [Continue with All Blocks...] to continue normally.

Replace​

The [Replace] option lets you choose to either replace a certain block ([Normal]), or replace all blocks except a certain block ([Exclude]). Once enabled, replacing can be disabled by clicking the button again.
